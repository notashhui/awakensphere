📖 JSON 格式说明（让 AI 输出此 schema 即可） 
[
  {
    "type": "article",            // 必填: article / podcast / video
    "title_zh": "中文标题",        // 必填
    "title_en": "English Title",
    "summary_zh": "一句话摘要",
    "summary_en": "One-line summary",
    "body_zh": "中文 Markdown 正文",
    "body_en": "English markdown body",
    "author": "作者名",
    "duration_zh": "8 分钟",
    "duration_en": "8 min",
    "category_zh": "正念冥想",
    "category_en": "MINDFULNESS",
    "tags": ["标签1", "标签2"],
    "cover_url": "https://...",
    "audio_url": "（仅 podcast，外链 mp3）",
    "podcast_link": "（仅 podcast，外部平台 URL）",
    "audio_embed": "（仅 podcast，iframe HTML）",
    "video_embed": "（仅 video，iframe HTML）",
    "guest_name": "（可选，嘉宾名）",
    "transcript": "（可选，文字稿）",
    "publish_date": "2026-05-07",
    "active": true,
    "featured": false
  }
]
· 单条用对象 / 多条用数组 — 都接受
· 必填字段：type + title_zh，其他全可省
· id 自动生成（c+时间戳），不要在 JSON 里指定
· sort_order 自动递增（不要指定）
· 上传后会预览，确认无误再提交
