### H-2：剩余单语字段双语化 ⭐ 本次完成

|---|---|---|
| 服务描述 | admin → 疗愈师 → 服务列表 → 描述 | 详情页"服务"区 |
| 活动地点 | admin → 疗愈师 → 活动日程 → 地点 | （前台未接通，admin 已存）|

**怎么测**：

2. **填一条英文**给一个服务的描述，保存后到前台 `healer.html?id=<那师id>`，切到 EN，看英文是否显示。
3. **不填英文**保存，切到 EN 看是否 fallback 到中文（不应该空白）。

### H-3：后台真分级导航 ⭐ 本次完成

**问题**：admin 后台所有疗愈师/内容垂直堆叠成一长串，找东西要滚很久。

**改动**：
- **列表视图（默认）**：精简卡片网格 — 头像/封面 + 名字 + 一句话职称 + 「编辑→」按钮 + 删除/上下移按钮。一屏可见 8-12 张。
- **编辑视图**：点列表卡片或左侧二级条目 → 进入独立编辑页（hash URL `#/healer/zoe`），完整字段表单。
- **顶部面包屑**：`← 返回疗愈师列表 / 当前师名`
- **底部独立保存**：原本就有的「💾 保存此疗愈师」按钮（编辑视图保留它，全量"保存所有"按钮在编辑视图自动隐藏，避免误操作）。
- **后退按钮可用**、刷新页记住位置（hash 路由的好处）

**怎么测**：
1. `admin.html` → 疗愈师管理 → 看到精简卡片网格，每张卡有头像 + 名字 + 职称 + 编辑→ + 删除×。
2. 点其中一张卡 → 进编辑视图，看顶部「← 返回疗愈师列表 / xxx」。地址栏 URL 应变成 `admin.html#/healer/<id>`。
3. 编辑某个字段（比如改个名字）→ 点底部「💾 保存此疗愈师」 → 等"保存中…"变绿色"已保存"。
4. 点顶部「← 返回疗愈师列表」→ 回到网格视图。
5. 浏览器后退按钮 → 应回到刚才编辑的那条。
6. 把地址栏 `admin.html#/healer/zoe` 直接复制粘贴打开 → 应直接进 zoe 的编辑页。
7. 内容库同样测一遍（点一篇文章 → 编辑 → 返回）。
8. **新增**：点「＋新增疗愈师」填完 → 应自动进入新建师的编辑页（continue editing）。

### J-1：用户账户 + 收藏功能 ⭐ 本次完成

**改动**：完整账户系统 + 跨设备同步收藏（不是 localStorage 临时方案，是 Supabase Auth + favorites 表）。

#### 测试 1：未登录加心形（本地 fallback）

1. 用**隐私窗口**打开 https://awakens.life（避免 cookie 干扰）
2. 看右上角 nav 末尾出现「登录 / 注册」按钮
3. 滚到内容区 → 每张文章/播客/视频卡片**右上角**有圆形心形按钮 ♡
4. 点一下心形 → 立即变 ♥（红色）+ 有个小弹动动画
5. 切到 EN，再切回 ZH → 心形状态应保留
6. 关闭隐私窗口，重新开 → ♥ 状态仍在（localStorage 持久）

#### 测试 2：去 healers 页测心形

1. https://awakens.life/healers.html
2. 每张疗愈师卡片右上角同样有 ♡ 按钮
3. 点 → 变 ♥
4. 进 healer 详情页 → 联系方式 pills 区第一个就是「♡ 收藏」（如果之前没收藏）或「♥ 已收藏」
5. 点切换状态

#### 测试 3：去 article 详情页测

1. 首页点任一文章/播客/视频 → article.html
2. 标题下方 byline 之后有一行胶囊形「♡ 收藏此内容」按钮
3. 点切换

#### 测试 4：注册账号

1. 右上角点「登录 / 注册」
2. 弹出 modal — 填邮箱 + 密码（至少 6 位）
3. 点最下面「没有账号？注册」切到注册模式
4. 填完点「注册」
5. **如果 Supabase Auth 设置了 confirm email**：会显示「请去邮箱激活账号」 — 你需要打开邮箱点确认链接才能登录
6. **如果没设 confirm email**：直接登录成功，modal 关闭，右上角变成「♥ 你的邮箱前缀」

#### 测试 5：登录 + 老收藏自动迁移

1. 用之前测试时未登录加过的浏览器 → 右上角「登录 / 注册」 → 用刚注册的账号登录
2. 登录成功后 → 右上角变成 「♥ 用户名」
3. **关键**：刚才未登录时本地加的所有收藏 → 应该自动同步到你的账号（首登触发迁移）
4. 点右上角「♥ 用户名」→ 跳到 me.html
5. 看到「我收藏的疗愈师」+「我收藏的内容」两个网格 → 应该包含你之前未登录时点过的那些

#### 测试 6：me.html 个人中心

1. 已登录状态打开 https://awakens.life/me.html
2. 顶部欢迎语「你好，<昵称>」+ 邮箱 + 「退出登录」按钮 + 「继续探索 →」
3. 「我收藏的疗愈师」section：每张卡有头像 + 名字 + 职称 + 「查看」（跳详情）+「取消」（红色）
4. 「我收藏的内容」section：每张卡有封面 + 类型徽章（文章/播客/视频）+ 标题 + 作者 + 时长 + 「查看」+「取消」
5. 点「取消」→ 该卡片立即从列表消失（且对应的心形按钮在其他页同步变 ♡）
6. **空态测试**：把所有收藏都取消 → 应该显示「还没收藏任何...」+「去探索 →」CTA

#### 测试 7：跨设备同步

1. 用手机打开 https://awakens.life/me.html
2. 用你刚注册的账号登录
3. **应能看到**电脑上加的所有收藏（这就是 Supabase 同步）

#### 测试 8：退出登录

1. me.html 点「退出登录」
2. 应回到未登录状态（右上角变回「登录 / 注册」按钮）
3. 心形状态切到本地 localStorage（如果之前是登录态加的，登出后理论上看不到）

#### 测试 9：跨标签同步

1. 开两个标签：A = me.html（已登录），B = healers.html（已登录）
2. 在 B 上加心形 → 应该立即同步到 A 的收藏列表（Supabase 实时）
3. 在 A 取消 → B 的心形也应变回 ♡（刷新一下）

#### 测试 10：RLS 隔离

1. 注册一个**第二个**账号（用另一邮箱）
2. 登录第二个账号 → me.html
3. **应该看不到第一个账号的收藏**（数据隔离 by RLS）
