## 1. 登录与基本概念

### 进入后台

浏览器打开 **`awakens.life/admin.html`**，输入邮箱 + 密码登录。

### 数据保存机制

- 你在后台的所有改动都直接保存到云端数据库（Supabase）
- 改动保存后，前台 awakens.life 上的访客**最长 10 分钟**内能看到（有缓存层降低费用）
- 你想立刻看到效果：在前台地址栏后加 `?nocache=1`，例如 `awakens.life/?nocache=1`

### 中英文双语

- 几乎所有字段都有"（中文）"和"（英文）"两个版本
- 网站访客点右上角"中文 / EN"切换，会自动显示对应语言
- 英文为空时，前台会用中文兜底显示（不会留空）

## 3. 疗愈师管理

### 新增疗愈师

1. 顶部点 **「＋ 新增疗愈师」**
2. 弹窗里填基础信息：**英文 ID**（必填，小写英文，如 `zoe` / `carrie`，**之后不能改**）+ 中文名 + 英文名 + 职称等
3. 点"添加"——疗愈师创建成功，可以继续编辑详细信息

### 字段说明

每位疗愈师卡片展开后能填的字段（按顺序）：

| 字段类别 | 字段 | 说明 |
|---|---|---|
| 基础 | 头像 | 点头像方框上传图片，会弹裁剪框（1:1 方形） |
| 基础 | 名字（中/英） | 显示在卡片顶部 |
| 基础 | 职称 / 副标题 / 职称补充 | 三个职称字段，最多三段（前台用 · 拼接） |
| 基础 | 简介 / 完整简介 | 简介=卡片摘要；完整简介=详情页 About 区 |
| 基础 | 价格 / 所在地 / 可约状态 / 预约日期 | 显示在卡片副信息区 |
| 分类 | **所属分类**（多选 chips） | 点 chip 选中（绿底）/取消，决定目录页筛选时是否被选中 |
| 分类 | **细标签** | 自由文本，按 Enter 添加。显示在卡片上的小标签 |
| 📋 个人档案 | 来自城市 / 国籍代码 / 出生年份 | 详情页左侧档案面板"From" 区 |
| 📋 个人档案 | 语言 / 执业开始年份 | 详情页档案面板对应行 |
| 📋 个人档案 | 认证列表（点 ＋ 添加多条） | 每条 = 认证名 + 年份 |
| 📞 联系方式 | 微信 / WhatsApp / 邮箱 / 小红书 / 预约链接 | **填了的字段才会在详情页显示对应按钮**；都不填则联系区隐藏 |
| 💬 特色引言 | 引言（中/英） + 引言来源说明 | 详情页底部大字摘录区 |
| 服务列表 | 多条服务（每条=名称+描述+价格+时长+图片） | 详情页"我的服务"区 |
| 活动列表 | 多条活动 | 详情页"近期活动"区 |
| 产品列表 | 多条产品 | 当前**前台暂时隐藏**整个产品区 |
| SEO | （已删除） | 此前的 og_image 等字段已移除 |

### 保存改动

每次编辑完，**滚到页面最底部点「💾 保存所有疗愈师」**。一次会保存所有疗愈师的改动。保存后会显示"✅ 已保存"。

> ⚠️ 切到别的菜单或刷新页面前**一定要保存**，否则改动会丢失。

### 删除疗愈师

每张卡片右上角有红色"删除"按钮。会弹确认框，二次确认后删除。**删除不可恢复**——如果只是不想显示，关掉"上线显示"开关即可。

## 5. 内容库

支持 **3 种内容格式**：📖 文章 / 🎙 播客 / ▶ 视频。

### 单条新增

1. 顶部点 **「＋ 新增内容」**
2. 弹窗填**类型**（文章 / 播客 / 视频 / 引导冥想）+ 标题 + 摘要 + 作者 + 时长
3. 点"添加" → 新条目出现在列表底部

### 编辑详细字段

每个内容卡片打开后，**根据"内容类型"会显示不同字段**：

| 字段 | 文章 | 播客 | 视频 |
|---|---|---|---|
| 标题 / 摘要 / 作者 / 时长 | ✅ | ✅ | ✅ |
| 封面图（带裁剪） | ✅ | ✅（方形） | ✅（16:9） |
| 分类（中/英） | ✅ | ✅ | ✅ |
| 📝 正文（中/英 Markdown） | ✅ | ✅ 当 show notes 用 | ✅ 当 description 用 |
| 🎵 音频文件 + 外部收听链接 + 嵌入代码 | ❌ | ✅ | ❌ |
| ▶ 视频嵌入代码 | ❌ | ❌ | ✅ |
| 嘉宾名称 / 发布日期 / 文字稿 | ❌ | ✅ | ✅ |

### 嵌入代码怎么获取

**🎙 播客嵌入代码**（推荐先发到小宇宙）：
1. 把节目发布到小宇宙（中文听众优先）/ Spotify / Apple Podcasts
2. 节目页面 → 分享 → **嵌入代码**
3. 复制完整的 `<iframe src="..." ...></iframe>` 整段
4. 粘贴到 admin 的"🎙 嵌入代码" textarea

**▶ 视频嵌入代码**：
1. 把视频发布到 B 站（中文）/ YouTube（国际）
2. 视频页面 → 分享 → 嵌入代码
3. 复制 `<iframe src="//player.bilibili.com/player.html?..." ...></iframe>`
4. 粘贴到 admin 的"▶ 视频嵌入代码" textarea

### 保存

底部 **「💾 保存内容库」**。

## 6. 批量导入

如果你用 AI（ChatGPT / Claude）批量生成多篇内容，可以一次导入。

### 操作步骤

1. 在内容库页顶部点 **「📂 批量导入（JSON）」**
2. 弹窗里点"📖 JSON 格式说明"看 schema（也可参考下方 [AI 提示词模板](#11-ai-提示词模板)）
3. 粘贴 JSON 到 textarea，**或**点"📁 从文件读取"上传 .json 文件
4. 点 **「🔍 预览」**——会显示：
   - ✅ 解析通过：N 条
   - ⚠ 错误清单（哪些条不合法，自动跳过）
   - 前 5 条预览
5. 检查无误，点 **「✓ 确认导入」** → 一次性插入数据库
6. 完成后会弹"✅ 批量导入成功：N 条"

### 必填字段

只有 2 个：
- `type`（必须是 article / podcast / video / meditation 之一）
- `title_zh`（中文标题）

其他都是可选——填了就用，没填就空着。

### 注意事项

- **不要在 JSON 里指定 `id` 或 `sort_order`** ——系统自动生成
- 如果某条已经存在（同 ID），会**新增**而不是覆盖（因为 ID 自动生成时永远是新的）
- 想改现有内容请用列表编辑，不要重复导入

## 7. 批量导出

### 用途

- **备份**：定期下载一份 JSON 留底
- **迁移**：把网站搬到别处时用
- **批量改完再导回**：导出后用文本编辑器批量替换内容，再用导入功能一次性更新（注意：导入是新增，先删旧的再导入）

### 操作步骤

1. 内容库页顶部点 **「📤 批量导出（JSON）」**
2. 浏览器自动下载文件 `awakensphere-content-2026-05-07.json`
3. 文件里是数组，每个对象一条内容（不含 id 和 sort_order，可以直接重新导入）
4. 用文本编辑器（VS Code / Sublime / 甚至记事本）打开查看 / 修改

### 文件结构示例

```json
[
  {
    "type": "article",
    "title_zh": "一杯茶的厚度",
    "title_en": "The Weight of a Cup of Tea",
    "summary_zh": "仪式不是迷信。",
    "body_zh": "## 仪式\n\n仪式不是...",
    "author": "Carrie Yang",
    "duration_zh": "8 分钟",
    "category_zh": "正念",
    "tags": ["仪式", "日常"],
    "cover_url": "https://res.cloudinary.com/...",
    "active": true,
    "featured": false
  },
  ...
]
```

## 8. 分类标签 / 疗愈方向

容易搞混的两个东西：

### 🏷️ 分类标签

首页顶部那条跑马灯：**"N°01 · 正念冥想 ◆ N°02 · 身体疗愈 ◆ ..."**

- 进 **「🏷️ 分类标签」** 菜单
- 输入框输入新标签 + 回车 / 点"添加"
- ▲▼ 排序、× 删除
- 底部 **「💾 保存标签」**

### 📁 疗愈方向

疗愈师目录页（healers.html）顶部的筛选 chip：**"全部 / 按摩 / 正念 / 声音 / 创伤 / 动态"**

- 进 **「📁 疗愈方向」** 菜单
- 每行可以改"中文标签"和"英文标签"（**slug 不能改**——它是代码识别用）
- ▲▼ 排序、× 删除
- 点"+ 新增分类"加新方向
- 底部 **「💾 保存所有分类」**

> 💡 在疗愈师卡片里选"所属分类"chips 时，看到的就是这里维护的列表。

## 9. 图片上传与裁剪

### 何时遇到

- 疗愈师头像
- 内容封面图
- 服务图片
- 首页 hero 视频海报

### 操作

1. 点 **"📷 从本地上传"** 或直接点头像方框
2. 选择图片文件（**最大 5MB**）
3. 弹出 **裁剪框**：
   - 头像 = 1:1 正方形
   - 内容封面 = 16:9 横屏
4. 拖动调整裁切区域 / 滚轮缩放
5. 点 **「✓ 确认并上传」**
6. 自动上传到 Cloudinary CDN，URL 写入数据库

### 提示

- 推荐尺寸：**头像至少 800×800**，**封面至少 1280×720**
- 文件大小**控制在 2MB 以内**最佳（更快加载）
- 输出自动是 JPEG 92% 质量

  "audio_embed": "（仅 podcast，完整 iframe HTML）",

不要在 JSON 里指定 id 或 sort_order（系统自动生成）

## 12. 常见问题 FAQ

### Q1：我改了字段保存了，前台为什么没变？

- 前台有 **10 分钟缓存**，正常情况下 10 分钟内会刷新
- 想立刻看：浏览器地址栏后加 `?nocache=1`，比如 `awakens.life/?nocache=1`
- 或者你保存后访问会**立刻清缓存**（admin 的保存自动触发）——但你需要刷新一次前台才生效

### Q2：保存失败提示"Could not find the 'X' column"

- 数据库 schema 落后于代码
- 联系开发去 Supabase SQL Editor 跑迁移文件

### Q3：上传图片显示"上传失败"

- 检查文件是不是超 5MB
- 检查网络（Cloudinary 是国外服务）
- 重新选择图片再试

### Q4：删除疗愈师 / 内容了，能恢复吗？

**不能**。删除是永久的。只是想下线请用"上线显示"开关。

### Q5：忘了保存就刷新了页面，改动还在吗？

不在。每次刷新会从数据库重新加载，未保存的改动会丢失。**改完一定保存**。

### Q6：批量导入后发现错了，怎么撤销？

没有撤销按钮。你需要：
1. 列表里找到那些导入的内容（最新的在底部）
2. 一条条点"删除"
3. **建议**：导入前先用"📤 批量导出"备份当前内容，出错时用导出的文件重新导入恢复

### Q7：怎么知道网站现在状态？

- Supabase Dashboard 看用量：[supabase.com/dashboard](https://supabase.com/dashboard)
- Netlify 部署状态：[app.netlify.com](https://app.netlify.com)
- 直接打开 [awakens.life](https://awakens.life) 看就完了

### Q8：我能给开发提需求吗？

可以。但请：
1. 写明**具体页面 / 区域**（截图最好）
2. 说明**期望效果**（不只是"现在不好看"）
3. 区分优先级（必须修 / 可优化 / 锦上添花）
4. 用 .docx 文件整理好统一发，比零散一句一句聊高效
